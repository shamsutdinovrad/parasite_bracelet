# Браслет, анализирующий речь на наличие слов-паразитов

Была поставлена задача создать алгоритм, способный понять, является ли окружающий звук речью владельца и, если да, найти в ней слова-паразиты и просигнализировать о их нахождении отправкой сигнала на микроконтроллер. 

Для распознавания, принадлежит ли речь владельцу, используется RandomForestClassifier из sklearn. Для обучения был собран небольшой датасет моего и чужих голосов в разных помещениях для большей стабильности. При обработке из записанного звука извлекаются MFCC и спектральные признаки, уже на них обучается классификатор. Для интересующихся, модель приложена в файле bracelet_model.pkl. Так же в файле bracelet_model.py приложен код, с помощью которого можно воспроизвести результаты, обучив модель уже на своем голосе.

Для speech-to-text распознавания используется модель vosk-model-small-ru-0.22, обученная конкретно для русского языка. Ссылка на модели: https://alphacephei.com/vosk/models

Может показаться, что код анализа и распознавания речи неоптимален из-за излишней конвертации raw файлов в wav, ведь как и собственную модель сразу можно обучить на raw файлах, так и использовать их в vosk, и тем самым отказаться от лишних файловых манипуляций в пользу буфера. Я пытался это реализовать, но постоянно получал нерабочие модели. В связи с этим было решено оставить не самое быстрое, но рабочее решение, тем более время, тратящееся на запись в файлы, никак не сказывается на качестве распознавания. Код обработки речи с браслета приложен в файле bracelet_main.py.

Код для микроконтроллера так же представлен в файле esp32.ino
